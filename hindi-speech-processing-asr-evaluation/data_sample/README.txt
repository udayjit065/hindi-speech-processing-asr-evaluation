Sample Data Folder
==================

This folder contains a small subset of the full dataset used for development and testing.

These are example files only and do NOT represent the complete dataset.
The actual project uses much larger speech corpora for training and evaluation.

Files:
- dataset.csv: Small sample of Hindi speech dataset with metadata

Note: Audio files (.wav, .mp3, etc.) are not included in the repository 
for storage efficiency. They can be downloaded separately.
